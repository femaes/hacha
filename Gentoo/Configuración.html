RED

Definir la variable hostname con el nombre elegido para el equipo
#nano -w /etc/conf.d/hostname
hostname="nombre elegido para el equipo"

El fichero /etc/conf.d/net no existe. hay que crearlo

Para IP estática
#nano -w /etc/conf.d/net
config_eth0="192.168.1.2 netmask 255.255.255.0 brd 192.168.1.255"
routes_eth0="default via 192.168.1.1"

Para IP dinámica por dhcp
#nano -w /etc/conf.d/net
config_eth0="dhcp"

Inicio automático de red en arranque
#cd /etc/init.d
#ln -s net.lo net.eth0
#rc-update add net.eth0 default

Si no funciona con eth0:
Actualizar el archivo /etc/conf.d/net indicando el nombre correcto de la interfaz (como enp3s0 en lugar de eth0).
Crear un nuevo enlace simbólico (como /etc/init.d/net.enp3s0).
#ln -s net.lo net.enp3s0
Eliminar el enlace simbólico antiguo (rm /etc/init.d/net.eth0).
Añadir el nuevo enlace al nivel de ejecución por defecto (default).
Eliminar el enalce anterior con rc-update del net.eth0 default.

LOCALIZACION

Verificar que las localizaciones seleccionadas están disponibles con la orden
$locale -a
Se puede mostar el fichero /usr/lib/locale/locale-archive usando la orden localedef.
$localedef --list-archive


Añadir localizaciones a

#nano -w /etc/locale.gen

en_GB ISO-8859-1
en_GB.UTF-8 UTF-8
de_DE ISO-8859-1
de_DE@euro ISO-8859-15

Actualizar
#locale-gen

Nota
Utilizar un valor @euro de /usr/share/i18n/SUPPORTED/ como localización cuando se utilice el símbolo monetario del Euro (€) en localizaciones que no estén basadas en UTF-8.

#eselect locale list
#eselect locale set 1

Definir la localización por defecto del sistemaen /etc/env.d/02locale

LANG="es_ES.UTF-8"
LC_COLLATE="C"

Para una localización por defecto para todo el sistema:
#env-update && source /etc/profile
